국기 이미지 (선택 사항)
=====================

언어 선택 화면에서 국기 이미지를 쓰려면 이 폴더에 아래 파일명으로 넣어 주세요.
PNG, JPG, SVG 모두 사용 가능합니다. (우선순위: .png → .jpg → .svg)
없으면 이모지 국기(🇺🇸 🇰🇷 등)가 표시됩니다.

파일명 예시 (하나만 있어도 됨):
  en.png / en.jpg / en.svg   - English (미국)
  ko.png / ko.jpg / ko.svg   - 한국어
  fr.png / fr.jpg / fr.svg   - Français (프랑스)
  ja.png / ja.jpg / ja.svg   - 日本語 (일본)
  zh.png / zh.jpg / zh.svg   - 简体中文 (중국)
  zh-TW.png / zh-TW.jpg / zh-TW.svg  - 繁體中文 (대만)

권장: 디자인이 이쁘게 보이려면 가로형 비율(예: 4:3)의 PNG 또는 JPG를 사용하세요.
